Les flexboxs

Ceci est un cours sur les flexboxs, pour un cours sur la navigation cliquez ici

Par défaut, le flux du document est orienté de haut en bas. Les éléments, s'additionnent verticalement

Si on veut disposer les éléments de manière horizontale, il est possible d'utiliser le display:flex sur le parent des éléments à mettre horizontalement

On peut choisir la direction de la disposition flex ( flex-direction) , ici un column-reverse

Ici , un row-reverse

Il existe plusieurs dispositons de centrage, comme space-around, space-between, start, end etc ... Ici le lien de la documentation pour retrouver les possibilités et/ ou éssayer

Pour centrer sur l'axe secondaire, il faut utiliser la propriété align-items. Si je suis en direction row ( rangée ) l'axe principale sera horizontal, l'axe secondaire vertical

Si il y a trop d'éléments, la page va s'aggrandir, ou le contenu va rapetissir, pour éviter cela, et avoir un retour a la ligne, on peut utiliser la propriété flex-wrap.
